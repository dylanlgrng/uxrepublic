# uxrepublic (GitHub Pages)

Ce repo est prêt pour être publié sur :
`https://dylanlagrange.fr/uxrepublic`

## Étapes (ultra simple)

1. Crée un repo GitHub nommé **uxrepublic**
2. Dépose le contenu de ce zip à la racine du repo
3. Dans GitHub → **Settings → Pages**
   - Source : **GitHub Actions**
4. Fais un commit sur la branche **main**
5. Attends que l’action **Deploy to GitHub Pages** passe en vert

> Important : ne mets PAS de "Custom domain" sur ce repo.
Le domaine `dylanlagrange.fr` doit rester géré par ton site racine.
